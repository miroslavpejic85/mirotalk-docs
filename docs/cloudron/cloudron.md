# MiroTalk on Cloudron

![cloudron](../images/cloudron.png)

MiroTalk is fully compatible with [Cloudron](https://www.cloudron.io/), a platform that simplifies the deployment and management of web applications. Running MiroTalk on Cloudron allows organizations to host a secure, self-managed video conferencing solution with minimal setup.

### Why MiroTalk + Cloudron

Deploying MiroTalk on Cloudron provides a reliable, enterprise-ready solution. Cloudron handles updates, backups, and user management, reducing administrative overhead and ensuring a stable, secure environment. This integration allows IT teams to focus on collaboration and productivity rather than server maintenance.

### Key Benefits

*  **One-click deployment** – Install MiroTalk quickly without complex server configuration.
*  **Automatic updates and backups** – Maintain a secure, up-to-date environment.
*  **Integrated user management** – Utilize Cloudron’s authentication and team management features.
*  **App ecosystem** – Combine MiroTalk with other Cloudron-hosted applications for a comprehensive self-hosted platform.

### Installation

MiroTalk can be installed directly from the [Cloudron App Store](https://www.cloudron.io/store/index.html). For more information on Cloudron and its capabilities, visit the [official website](https://www.cloudron.io/).
